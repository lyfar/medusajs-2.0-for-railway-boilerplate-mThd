# Operational Guides

This section outlines the documentation that will be provided to ensure the team can manage the e-commerce operations effectively.

## Key How-To Guides

- **Product Management**: A step-by-step guide on how to add a new diamond to the catalog, including which fields are mandatory.
- **Data Management**: Instructions on how to run catalog import and cleanup processes.
- **Inventory Control**: A guide on how to perform inventory audits and ensure stock levels are accurate.
- **Troubleshooting**: Simple playbooks for resolving common operational issues.
