# SEO & Content

This section covers SEO foundation work starting from Phase 1 (technical setup) through Phase 3 (content strategy), with continuous implementation throughout development.

## SEO Implementation Timeline

| Phase | Technical SEO | Content Strategy | Implementation |
| --- | --- | --- | --- |
| **Phase 1 (Weeks 1-2)** | Site structure, URL planning, meta framework | Keyword research planning | Foundation setup |
| **Phase 2 (Weeks 2-5)** | Technical implementation, structured data | Content planning, brand voice | Continuous development |
| **Phase 3 (Weeks 5-6)** | Testing, optimization, analytics setup | Content framework, editorial process | Final implementation |

## Technical SEO Implementation

| Element | Purpose | Implementation |
| --- | --- | --- |
| **Site Structure** | Country routing, collections, categories | Clean URL hierarchy |
| **Product Pages** | Diamond listings, detail page optimization | Rich snippets, structured data |
| **Meta Framework** | Sitemap, robots.txt, social sharing | OpenGraph, Twitter cards |

## Minimal Analytics at Launch

| Tooling | Events | Notes |
| --- | --- | --- |
| **GA4** | view_item, add_to_cart, begin_checkout, purchase | Cookie consent banner and IP anonymization |
| **GTM (optional)** | Tag management | Keep minimal to reduce overhead |
| **Consent** | Banner + policy | Basic privacy policy and preferences |

## Content Strategy Preparation

| Strategy Component | Purpose |
| --- | --- |
| **AI-Assisted Content** | Long-form articles on lab-grown diamond topics |
| **Monthly AI SEO Publishing** | Aggressive, targeted content plan | Budget: ~8,000 HKD/month |
| **Editorial Process** | Brand voice guidelines and content review workflow |
| **SEO Content Plan** | Keyword targeting and content calendar framework |

**Note**: Content creation and ongoing SEO management are post-launch activities requiring separate ongoing engagement.
